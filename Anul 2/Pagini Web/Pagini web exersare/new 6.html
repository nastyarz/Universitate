Glosar de termeni de World Wide Web

hypertext 

 - o interconectare Web de informatii de tip text, in care orice 

cuvant sau fraza poate face legatura catre un alt punct din

document sau catre un alt document

date

 - fluxul nesfarsit de materiale care apar pe Internet,spre

deosebire de informatii care sunt date cu un anumit inteles

sau valoare

informatie

 - sub-setul de date care are efectiv si care este

util la momentul curent
